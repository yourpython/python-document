二，安装
首先认清自己的py版本，我的是py3.6
1,安装 openpyxl 
py(3.4之前都没有自带pip.exe，需要先安装pip.你先下载个pip安装包，然后解压至python文件夹中的Scripts，cmd cd 到该文件夹中的解压包，
输入：py setup.py install 即可)，所以自带pip,直接在win自带的dos,输入pip openyxl install 
2,安装 pandas 
需要先安装 wheel , 安装 ：pip wheel  install 
然后去官网下载 pandas.whl 注意：要记得对应版本，我的是py3.6 and win10 64 位 所以我下载的是pandas-0.20.0rc1-cp36-cp36m-win_amd64.whl
最后 pip pandas-0.20.0rc1-cp36-cp36m-win_amd64.whl install  即可。
